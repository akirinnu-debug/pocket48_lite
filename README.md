# Pocket48 Lite

轻量级第三方口袋48直播回放客户端。APK 不到 9MB，免登录、零权限、无广告。

- **回放** — 浏览 SNH48 集团各团成员的直播回放，支持倍速
- **弹幕** — 飞行弹幕 + 弹幕池面板
- **缓存** — ExoPlayer LRU 缓存（上限 2GB），二次观看无需重复加载
- **成员** — 按团体/队伍筛选，支持收藏关注

## 技术栈

Kotlin / Jetpack Compose + Material 3 / Navigation Compose / MVVM / OkHttp / kotlinx-serialization / Coil / Media3 ExoPlayer / DataStore
